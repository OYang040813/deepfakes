关于视频、图像与音频的深度伪造检测平台

- [贡献](#贡献)

## 安装&使用

```bash
# 克隆仓库
git clone https://github.com/OYang040813/deepfakes.git

# 进入项目目录
cd deepfakes

# 启动项目前端
cd vue
npm run serve

# 启动项目后端
cd ../springboot
mvn spring-boot:run
```
